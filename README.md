## Nest-App

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode (server keeps running and detects changes)
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Nest App Terminal Commands
```bash
# Install Nest
npm i -g @nestjs/cli  

# Create Nest App
nest new $project-name 

# Add module
nest g module $module-name

# Add Controller
nest g controller $controller-name

# Add Service
nest g service $service-name


```
